# Network Security Baseline Checklist

## Perimeter Security
- [ ] Edge router properly configured as default gateway
- [ ] NAT implemented correctly
- [ ] Unused services disabled on edge devices

## Firewall Controls
- [ ] Stateful firewall enabled
- [ ] Inbound rules restricted to required services
- [ ] Outbound traffic policies defined
- [ ] Firewall rule review process exists

## Network Segmentation
- [ ] User VLAN isolated from server VLAN
- [ ] Management VLAN restricted
- [ ] Trunk VLANs explicitly allowed
- [ ] No unnecessary broadcast domains

## Device Access & Management
- [ ] Secure management protocols in use (SSH)
- [ ] Telnet disabled
- [ ] Role-based access implemented
- [ ] Shared credentials minimized

## Monitoring & Visibility
- [ ] Logging enabled on key devices
- [ ] Logs retained for review
- [ ] Basic alerting in place

## Documentation
- [ ] Logical network diagram exists
- [ ] Physical network diagram exists
- [ ] VLAN and addressing documented
- [ ] Change history tracked
